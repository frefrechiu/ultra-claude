```python
class AgentConfig(BaseModel):

    role: str = Field(min_length=1, ...)
    adapter: Literal["claude", "gemini", "codex"]
    system_prompt: str = Field(min_length=1, ...)

class RoundtableConfig(BaseModel):
    agents: list[AgentConfig] = Field(min_length=2, ...)
```

```yaml
# Bundled preset: 3-agent debate roundtable.

      You design system architecture. Prioritize simplicity, explicit data flow,
      and minimal dependencies. Push back on over-engineering. Lead with constraints.

  - name: Critic
    role: skeptic
    adapter: gemini
    system_prompt: |
      You poke holes in proposed designs. Reference past production failures.
      Challenge assumptions. Ask "what happens when X fails?" Be ruthless but constructive.

  - name: Implementer
    role: hands-on coder
    adapter: codex
    system_prompt: |
      You write the actual code. Flag any unbuildable parts of the proposed design.

max_turns: 9
stop_keywords:
  - AGREED
  - SHIP IT
```

Notes on the content:

* No `transcript_path:` -- left unset so the orchestrator falls back to `ultra-claude-transcript.md` in cwd (or whatever `--output` overrides on the CLI).
* No `task:` -- the CLI requires the user to supply one via positional `TASK_FILE` or `--inline`.
* No `abort_on_error:` -- defaults to `false` per `RoundtableConfig` (continue-on-error per CLAUDE.md key decision #9).

Do NOT use Unicode characters (em-dashes, smart quotes, emoji) in the YAML even though the file is UTF-8 -- per CLAUDE.md Critical Constraint #6, ASCII-only on disk is the project standard so the file is identical across all editors and platforms.

# ultra-claude

Orchestrate Claude Code, Gemini CLI, and Codex CLI in a multi-agent debate — using your existing CLI logins. No API keys.

> **Note:** Independent open-source project. Not affiliated with Anthropic, Google, or OpenAI. "Claude" is a trademark of Anthropic; "Gemini" of Google; "Codex" of OpenAI. ultra-claude is a third-party orchestrator that invokes their respective official CLIs as subprocesses.

This is the **0.0.1 name-reservation stub**. The functional release ships as `0.1.0`. See [the GitHub repo](https://github.com/frefrechiu/ultra-claude) for status.
